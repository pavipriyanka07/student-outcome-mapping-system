# Student Academic Learning Outcome Mapping System

## Overview
The Student Academic Learning Outcome Mapping System is a web-based application developed to support Outcome-Based Education (OBE).

The system maps Course Outcomes (CO) with Program Outcomes (PO), records student academic performance, and calculates attainment levels for evaluation and accreditation purposes.

---

## Features
- Student Management
- Course Management
- Course Outcome (CO) and Program Outcome (PO) Management
- CO–PO Mapping (Level 1, 2, 3)
- Marks Entry
- CO and PO Attainment Calculation

---

## Tech Stack
Frontend: HTML, CSS, JavaScript / React  
Backend: Node.js, Express.js  
Database: MySQL  

---

## Architecture
Frontend → REST API (Express.js) → MySQL Database

---

## Key APIs
- GET /students  
- POST /students  
- POST /mapping  
- GET /attainment/co  
- GET /attainment/po
